# Recursos Curados por Área

Recursos gratuitos o de bajo costo seleccionados específicamente para este proyecto. Calidad sobre cantidad.

---

## Python y análisis de datos
- **Kaggle Learn** (kaggle.com/learn) — Cursos gratuitos de Python, Pandas, visualización y ML. Cortos, prácticos y con ejercicios reales.
- **Real Python** (realpython.com) — Tutoriales detallados para problemas específicos.
- **Python for Data Analysis** — Libro de Wes McKinney (creador de Pandas). Referencia esencial.

## Estadística
- **StatQuest with Josh Starmer** (YouTube) — El mejor canal para entender estadística y ML desde cero, con intuición visual. Completamente gratuito.
- **Statistics for Data Science** — Canal de Brandon Foltz en YouTube.

## Machine Learning
- **Scikit-learn User Guide** (scikit-learn.org) — Documentación oficial con ejemplos. La referencia más confiable.
- **fast.ai** (fast.ai) — Curso gratuito de deep learning enfocado en resultados prácticos primero.
- **Hands-On Machine Learning** — Libro de Aurélien Géron. El más recomendado para ML aplicado.

## Series de tiempo
- **Prophet Documentation** (facebook.github.io/prophet) — Directamente aplicable a pronóstico climático.
- **Forecasting: Principles and Practice** (otexts.com/fpp3) — Libro gratuito online, referencia académica de series de tiempo.

## Computer Vision
- **Ultralytics YOLOv8 Docs** (docs.ultralytics.com) — Documentación oficial de YOLO, con tutoriales paso a paso.
- **OpenCV Tutorials** (docs.opencv.org) — Referencia completa para procesamiento de imágenes.
- **Papers With Code** (paperswithcode.com) — Estado del arte en computer vision con código disponible.

## Análisis Geoespacial
- **Geographic Data Science with Python** (geographicdata.science/book) — Libro gratuito online, muy completo.
- **Google Earth Engine Guides** (developers.google.com/earth-engine) — Documentación oficial con ejemplos.
- **Folium Documentation** (python-visualization.github.io/folium) — Para mapas interactivos rápidos.

## Streamlit y dashboards
- **Streamlit Documentation** (docs.streamlit.io) — Aprende a construir una app en 30 minutos.
- **Storytelling with Data** (storytellingwithdata.com) — Blog y libro sobre cómo comunicar datos a no técnicos.

## Ingeniería Civil + Data Science (dominio combinado)
- **McKinsey Global Institute: Reinventing Construction** (disponible online) — Informe clave sobre digitalización en construcción.
- **KPMG: Global Construction Survey** — Tendencias globales en tecnología para construcción.
- **Construction Innovation Hub** (constructioninnovationhub.org.uk) — Recursos del hub del Reino Unido.

## Datos para Perú
- **SENAMHI** (senamhi.gob.pe) — Datos climáticos históricos y en tiempo real.
- **CENEPRED** (cenepred.gob.pe) — Mapas de riesgo de desastres naturales.
- **IGN — Geoportal** (geoportal.igm.gob.pe) — Cartografía oficial del Perú.
- **OSCE** (osce.gob.pe) — Datos de contratos de obras públicas.
- **INEI** (inei.gob.pe) — Datos socioeconómicos y geográficos.
- **GeoServidor MINAM** (geoservidor.minam.gob.pe) — Datos ambientales y geoespaciales.

## Comunidades en Perú
- **BIM Forum Perú** — Para conectar con el ecosistema de digitalización en construcción.
- **PMI Capítulo Perú** — Red de gestión de proyectos.
- **Data Science Perú** (LinkedIn/Meetup) — Comunidad local de ciencia de datos.
