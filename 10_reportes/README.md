# Reportes

Aquí se guardan los reportes y entregables generados para la empresa.

## Nomenclatura de archivos

```
YYYYMM_tipo_descripcion.pdf
Ejemplo: 202604_analisis_riesgo_climatico_obra_norte.pdf
```

## Tipos de reportes

- **analisis_** : Análisis técnico con metodología y resultados
- **dashboard_** : Capturas o exportaciones de dashboards
- **propuesta_** : Propuestas formales para gerencia
- **informe_** : Informes periódicos (semanales, mensuales)

## Regla importante

Los reportes aquí son solo los entregables finales. El código y datos que los generaron están en `07_proyectos/`.
