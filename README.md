# Data Science aplicado a la Ingeniería Civil
### Repositorio de conocimiento — Fernando Geronimo Ccoillor Soto

---

## Visión del proyecto

Este repositorio es un sistema de conocimiento vivo. Su propósito es construir una base sólida en la intersección entre **Data Science** e **Ingeniería Civil**, con aplicación directa al sector construcción en Perú — especialmente en infraestructura vial y de puentes.

No es solo un colección de código. Es un activo profesional de largo plazo que documenta teoría, herramientas, proyectos reales y conocimiento del dominio de forma estructurada, entendible y replicable en el tiempo.

La meta central es crear una nueva área dentro de la industria de la construcción peruana que integre análisis de datos, machine learning y visualización para tomar mejores decisiones en obra.

## Principios del repositorio

1. **Teoría conectada a práctica** — Cada concepto teórico está vinculado a un caso de aplicación en construcción.
2. **Replicable** — Cada proyecto incluye entorno, datos de ejemplo y documentación suficiente para reproducirlo.
3. **Creciente** — El repositorio escala con el tiempo. Cada proyecto nuevo sigue la misma estructura interna.
4. **Entendible sin código** — Los resultados siempre se traducen a valor de negocio: tiempo, dinero o riesgo.
